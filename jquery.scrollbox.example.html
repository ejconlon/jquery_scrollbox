jquery.scrollbox demo

by Eric Conlon

 

Tested on Firefox 3.5, Safari 4, and Opera 10 (though Opera doesn't animate very smoothly for some reason). The last time I tested it in IE7 it worked fine, but that was a few revisions ago... Please let me know what you find in IE6/7/8.

 

jquery.scrollbox is a jQuery extension that allows you to easily and discretely scroll content with few changes to your markup. For example, it could be used vertically on <p> elements: 

$("#bar").scrollbox({
	fadeClass: "fade",
	itemSelector: "#bar p",
	vertical: true,
	numShown: 1,
});

Here are some some options and their defaults. 

$("#defElemId").scrollbox({
	vertical : false,				
	// ^ true for horizontal
	 
	itemSelector : "#"+defElemId+" > *", 
	// ^ selector for those elements to scroll
	
	nextSelector : "."+defElemId+"_next",
	// ^ selector to set a trigger to scroll
	
	prevSelector : "."+defElemId+"_prev",
	
	numShown : 3,					
	// ^ how many things to show at once
	
	fadeClass : "fade", 
	// ^ what class to set next/prevSelector when scroll reaches end
	
	maxWidth : 10000000000000, 
	// ^ usually this will be calculated automagically
	
	maxHeight : 10000000000000
});
